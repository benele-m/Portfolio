Real problems I've worked on

Troubleshooting Notes

Short notes — problem, cause, fix, takeaway.

Hardware · Windows

Frequent BSODs

  • Problem

    Random Windows crashes under load.

  • Cause

    Faulty RAM stick + outdated GPU driver.

  • Fix

    MemTest86, removed bad stick, clean driver reinstall.

  • Takeaway

    Learned to read stop codes and use Event Viewer.

Backend · Database

DB Connection Timeouts

  • Problem

    500 errors and slow API requests under load.

  • Cause

    PostgreSQL connection pool was exhausted.

  • Fix

    Tuned pool size, added query timeouts, added a missing index.

  • Takeaway

    Got a better feel for connection pooling.

Networking · LAN

Random LAN Drops

  • Problem

    Devices kept losing internet at random.

  • Cause

    Misconfigured DHCP scope handing out conflicting IPs.

  • Fix

    Reset DHCP range, longer leases, static reservations for key devices.

  • Takeaway

    Clearer mental model of how DHCP actually works.

Security · Linux

SSH Brute-Force Attempts

  • Problem

    Logs filling with failed SSH logins from random IPs.

  • Cause

    Default port + password auth + no rate limiting.

  • Fix

    Key-based auth, disabled root login, installed fail2ban.

  • Takeaway

    Hands-on intro to basic server hardening.

Web Development

CORS Error on Contact Form

  • Problem

    Browser blocked requests with a CORS error.

  • Cause

    Opening the project as file:// instead of via a server.

  • Fix

    Ran the project with Live Server.

  • Takeaway

    Learned why a real local server matters.
